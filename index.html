Mission

We want to collect, curate & open source the data which is presently only available to big companies. This data can be used for fueling research, however the companies sell the data, becoming a roadblock for research. To catalyze the process we’ve crafted an android application that records the data used by a person on wifi & cellular network. This data is pushed on the server where it is collected & open sourced for further analytics.
